## Installation And Usage

To run this project run

```bash
  git clone https://github.com/Pallavi3010/E-mart.git
```
Go to directory
```bash
  cd e-mart
```
Install all dependecies
```bash
 npm install
```
Run Project
```bash
  npm start
```

Now you can access  on localhost:3000
